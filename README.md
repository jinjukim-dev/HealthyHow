# HealthyHow 건강How
### 🥗 2018 2학기 javaWeb 수업 프로젝트, 건강 레시피를 공유하는 웹

> 여러 인기 tv프로그램에 나온 레시피를 한눈에 보고, 직접 레시피를 올려서 공유하는 웹사이트

### 기능
- 회원가입 / 로그인 기능
- 마이페이지 (내가 쓴 글 모아보기) 기능
- 레시피 작성 기능
- 카테고리별 레시피 보는 기능
- 게시글 작성 / 보는 기능
